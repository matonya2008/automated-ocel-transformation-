# Release Steps

1. Create a new GitHub repository under `matonya2008`.
2. Copy the contents of `public_ocel_code_release/` into that repository.
3. Review `docs/SANITIZATION_CHECKLIST.md`.
4. Commit the sanitized code and documentation.
5. Push the repository and create a versioned tag, for example `v1.0.0`.
6. Connect the repository to Zenodo and mint a DOI from the tagged release.
7. Replace the placeholder public-code statement in the manuscript with the final GitHub URL and Zenodo DOI.
